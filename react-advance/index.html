Why we need react? what facebook cannot able to do, they need to build react?

Facebook was facing problem with notification and reaction that were not changing in real time, there engineer build a library react. Before react they were using broswer DOM which was creating problem if we make any changes all the DOM Tree get reloaded, which were creates a problem so facebook/ react engineer develop their own dom called virtual DOM, same like browser DOM, but that is slightly different, when virtual dom get refreshed, it only changes the current changes in the dom tree, instead whole DOMTree, which is effective way. This is reason to use react for big applications.
